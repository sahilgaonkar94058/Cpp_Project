BASIC LINUX COMMANDS


*INFORMATION ABOUT LINUX*

LINUX is an operating system or a kernel distributed under an open-source license. Its functionality list is quite like UNIX. The kernel is a program at the heart of the Linux operating system that takes care of fundamental stuff, like letting hardware communicate with software.


  • Cal

    To open calender

  • Date

    Display date

  • Jobs

    Display running jobs / programs

  • fg<ProcessId>

    Start paused process

  • <command/process>&

    & will run command in background

  • which<FileName>

    Gives path of program/code file

  • gedit <command/program>

    Gedit is text editor it will open file with given name

  • mkdir <DirName>

    It will create directory with given name

  • rmdir <DirName>

    It will remove directory with given name

  • cd <FolderName>

    Change directory with given folder name or folder path

  • cat <fileName>

    Display content inside file

  • free -m

    shows memory(RAM) usage

  • history

    displays previously executed commands



*INFORMATION ABOUT LVM*

In Linux, Logical Volume Manager is a device mapper framework that provides logical volume management for the Linux kernel. Most modern Linux distributions are LVM-aware to the point of being able to have their root file systems on a logical volume


LVM COMMANDS


  • fdisk -l

  • pvcreate DiskName

  • pvdisplay

  • vgcreate disk1 disk2

  • vgdisplay

  • lvcreate --size (SizeINnumber)G --name lvname vgname

  • lvdisplay

  • mkfs.ext4 /dev/(vgname)/(lvname)

  • mount /dev/(vgname)/(lvname) (MountPointFolder)

  • resize2fs /dev/(vgname)/(lvname)



*INFORMATION ABOUT DOCKER*

Docker is a set of platform as a service products that use OS-level virtualization to deliver software in packages called containers. Containers are isolated from one another and bundle their own software, libraries and configuration files; they can communicate with each other through well-defined channels
